# solar-tracker
A small turntable that tracks the sun

Based on a thingiverse print: https://www.thingiverse.com/thing:53321

# Electronics

I have some old PIC16F690 lying around so the circuitry is based on that.

# Building and making

install `sdcc` and `pk2cmd`

```
make
```

```
make program
```

```
make run
```
